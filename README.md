# Student Performance Regression

Ce projet analyse les facteurs qui influencent la note finale d’élèves.

## Objectif
Prédire la note finale (G3) à partir de plusieurs variables comme :
- temps d’étude
- absences
- échecs précédents
- notes intermédiaires

## Méthode
Modèle de régression linéaire réalisé avec Python (pandas, scikit-learn).

## Résultat
Les notes intermédiaires (G1 et G2) apparaissent comme les variables les plus fortement liées à la note finale.
